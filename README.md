Intelscope v1.0 🛡️
Intelscope is a C++ security simulation tool that analyzes password strength through policy enforcement and brute-force simulation.

## Features
- Real-time Analysis: Uses `windows.h` for high-precision timing.
- UI Feedback: Dynamic color changes based on security status.
- Audible Alerts:System beeps when vulnerabilities are detected.
- Performance:Optimized loop logic with modulo-based UI updates.

## Technical Details
- Language: C++
- Libraries: `iostream`, `string`, `windows.h`, `ctime`
